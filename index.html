Способи використання JavaScript:

1. JavaScript у тегу <script> всередині HTML

Дивіться результат у консолі.

2. Підключення зовнішнього файлу

Цей спосіб дозволяє тримати JavaScript-код в окремому файлі. Наприклад: 

<script src="script.js"></script>

У цьому прикладі файл script.js повинен знаходитися в тій самій папці, що й HTML-файл.

3. Інлайн-скрипт

JavaScript вбудовується безпосередньо в атрибути HTML, наприклад onclick.
